# miniguia-estudos-gamificando-linguaptbr

Este repositório reúne **objetivos**, **fontes**, **testes de prompts** e um **mini guia prático** sobre como usar gamificação para melhorar os estudos em língua portuguesa (PT-BR).

---

## 🎯 Objetivo do projeto

- Explorar formas de **gamificar processos de estudo** em língua portuguesa.
- Testar **prompts de IA** para gerar materiais educacionais mais consistentes.
- Organizar **fontes e referências** que embasam o mini guia.
- Produzir um **guia enxuto e direto**, que qualquer pessoa consiga aplicar.

---

## 📁 Estrutura do repositório

A estrutura planejada (pode evoluir com o tempo):

- `objetivos.md`  
  A ideia principal é que gamificar não significa apenas usar jogos, mas sim criar uma experiência de aprendizagem mais engajadora, com propósito, progresso visível, feedback constante e desafios significativos.

- `fontes.md`  
  Artigos em PDF.

- `mini-guia.md`  
  Conteúdo principal com orientações práticas sobre estudos gamificados em PT-BR.

- `prompts/`  
  - `prompt-base.md` – prompt principal usado nos testes.  
  - `variacoes.md` – versões alternativas e ajustes.  
  - `exemplos.md` – exemplos de entrada (input) e saída (output).
